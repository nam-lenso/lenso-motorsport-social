Privacy Policy

Last updated: March 24, 2026

Overview

Lenso Motorsport Social is an internal analytics tool operated by Lenso Wheel Co., Ltd. It accesses Facebook and Instagram page data solely for the purpose of managing and analyzing the Lenso Motorsport social media presence.

Data We Access

This application accesses the following data from Meta platforms via the Graph API:

  • Page post metrics (likes, comments, shares, reach)
  • Page-level insights (follower count, impressions, engagement)
  • Instagram media and engagement metrics
  • Public competitor page information

How We Use Data

All data is used exclusively for internal content performance analysis and social media planning. Data is stored locally and is not shared with any third parties.

Data Storage

Data is stored in a local SQLite database on secured devices. No data is transmitted to external servers or cloud services.

Data Retention

Metrics data is retained locally for as long as needed for analysis purposes and can be deleted at any time.

Third-Party Access

This application does not share, sell, or transfer any accessed data to third parties.

Contact

For questions about this privacy policy, contact the Lenso Motorsport team via the Lenso Wheel company website.
